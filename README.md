# 一个tauri构建的音乐播放器

```bash
npm install -g pnpm

pnpm install

#一定要执行这个打包命令再运行tauri因为rust会报错
pnpm build

# 仅在rust环境下可用
pnpm tauri dev

# 浏览器环境运行
pnpm dev

```

第一次运行可能比较久，耐心等待，还需要rust环境和webview2  

没装rust环境可以照着配一下，windows7及以下可能没有webview，这个可以去下载一下
